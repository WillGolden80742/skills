# Skill Doc Co-Authoring

## Quando Oferecer

- Usuário menciona escrever documentação
- Tipos específicos: PRD, design doc, decision doc, RFC
- Tarefa substancial de escrita

## Workflow em 3 Estágios

### Estágio 1: Coleta de Contexto

**Meta**: Reduzir gap entre o que o usuário sabe e o que a IA sabe.

**Perguntas Iniciais**:
1. Que tipo de documento é este?
2. Quem é o público-alvo?
3. Qual impacto desejado?
4. Há template ou formato específico?
5. Outras restrições?

Depois, deixe o usuário despejar todo contexto que tiver.

### Estágio 2: Refinamento e Estrutura

**Meta**: Construir documento seção por seção.

**Para cada seção**:
1. Faça perguntas clarification
2. Brainstorm 5-20 opções
3. Usuário indica o que manter/remover
4. Draft a seção
5. Refine iterativamente

### Estágio 3: Teste com Leitor

**Meta**: Verificar se o documento funciona para leitores reais.

1. Preveja perguntas que leitores fariam
2. Teste com sessão fresca (sem contexto)
3. Identifique ambiguidades
4. Corrija lacunas

## Dicas

- Seja direto e procedural
- Dê agência ao usuário para ajustar o processo
- Priorize qualidade sobre velocidade
