# Skill Internal Comms

## Quando Usar

Para escrever comunicações internas:
- Atualizações 3P (Progresso, Planos, Problemas)
- Newsletters da empresa
- Respostas de FAQs
- Relatórios de status
- Atualizações para liderança
- Atualizações de projeto
- Relatórios de incidentes

## Como Usar

1. **Identifique o tipo** de comunicação
2. **Carregue o guideline** apropriado de `examples/`:
   - `examples/3p-updates.md` - Atualizações de Progresso/Planos/Problemas
   - `examples/company-newsletter.md` - Newsletters empresariais
   - `examples/faq-answers.md` - Respondendo FAQs
   - `examples/general-comms.md` - Para qualquer outro tipo
3. **Siga as instruções** específicas para formatação, tom e conteúdo
